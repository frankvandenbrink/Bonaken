# Bonaken Installeren zonder Play Store

## APK Bestand

Het APK bestand staat in:
```
client/android/app/build/outputs/apk/release/app-release.apk
```
Grootte: ~3 MB

## Versturen

Stuur het APK bestand via:
- WhatsApp
- Telegram
- Email
- Google Drive
- AirDrop (naar jezelf, dan doorsturen)
- USB kabel

## Installeren op Android

1. **Open het APK bestand** op de Android telefoon
2. Android toont een melding: "Installeren vanuit onbekende bronnen niet toegestaan"
3. Tik op **Instellingen** en schakel **Toestaan vanuit deze bron** in
4. Ga terug en tik op **Installeren**
5. Na installatie: tik op **Openen**

## Vereisten

- Android 5.0 (API 22) of hoger
- Internetverbinding (het spel verbindt met de server op bonaken.frankvdbrink.nl)

## Bijwerken

Bij een nieuwe versie: stuur het nieuwe APK bestand opnieuw en installeer het. De oude versie wordt automatisch overschreven.
